# Интеграция WebApp в Telegram Бот

Чтобы подключить этот WebApp к вашему Telegram-боту, выполните следующие шаги:

## 1. Развертывание WebApp
Сначала нужно разместить это приложение на хостинге, поддерживающем Next.js (например, Vercel).

1. Создайте репозиторий на GitHub и загрузите туда код.
2. Подключите репозиторий к Vercel.
3. Получите публичный URL вашего приложения (например, `https://cryptomastery-miniapp.vercel.app/`).

## 2. Настройка бота через BotFather
1. Откройте [@BotFather](https://t.me/BotFather) в Telegram.
2. Выберите вашего бота через `/mybots`.
3. Перейдите в **Bot Settings** -> **Menu Button**.
4. Нажмите **Configure Menu Button**.
5. Отправьте URL вашего WebApp, который вы получили на этапе 1.
6. Введите название кнопки, например, "Открыть приложение".

Теперь при нажатии на кнопку меню в вашем боте будет открываться WebApp.

## 3. Интеграция данных (Telegram WebApp API)
В текущей версии приложение использует моковые данные. Чтобы использовать реальные данные пользователя из Telegram:

1. В файле `webapp/lib/user-context.tsx` раскомментируйте или добавьте код для чтения `initData`:

```typescript
useEffect(() => {
  // @ts-ignore
  const tg = window.Telegram?.WebApp
  if (tg) {
    tg.ready()
    const user = tg.initDataUnsafe?.user
    if (user) {
      updateUser({
        id: user.id.toString(),
        nickname: user.username || user.first_name,
        // ... другие поля
      })
    }
  }
}, [])
```

2. Убедитесь, что скрипт Telegram WebApp подключен в `layout.tsx` (в `<head>`):
```html
<script src="https://telegram.org/js/telegram-web-app.js"></script>
```

## 4. Логика подписки и Менторов
В приложении реализована следующая логика:
- **Курсы**: Изначально список пуст. Вы можете добавлять их через Админ-панель (доступна, если у пользователя `isAdmin: true`).
- **Чаты**: 
  - Пользователю доступны 3 чата: 2 Ментора и Поддержка.
  - Писать в чат поддержки можно всегда.
  - Писать менторам можно **только при наличии подписки** на этого конкретного ментора.
  - Если подписки нет, поле ввода заблокировано и отображается сообщение "Требуется подписка".

## 5. Админ-панель
Для доступа к админ-панели:
1. В коде (`user-context.tsx`) временно установите `isAdmin: true` для вашего ID.
2. Либо реализуйте проверку на бэкенде: если `user.id` из `initData` совпадает с вашим ID админа, выдавайте права.
3. В профиле появится кнопка "Админ-панель", где можно добавлять курсы.
